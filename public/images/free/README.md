# Папка для изображений администратора

В эту папку кидайте все изображения, которые хотите использовать для редактирования сайта.

## Поддерживаемые форматы:
- PNG (с прозрачностью)
- JPG (фотографии)
- SVG (векторная графика)
- WebP (оптимизированные изображения)

## Как использовать:
1. Перетащите изображения в эту папку
2. Откройте админ-панель: `/admin_designer/?password=Booger999$`
3. В нижней панели нажмите на стрелку для раскрытия
4. Перетаскивайте изображения на элементы сайта

## Примеры изображений:
- `logo.svg` - для логотипа
- `hero-bg.jpg` - для фона главной секции
- `tool-icons/` - иконки для инструментов
- `social/` - иконки социальных сетей
